## About
Kiriapp - Online Course (Services)

## Installations
Make sure you already install [bun runtime](https://bun.sh/)
```bash
bun install
```

Configure <code>.env</code> and run <code>seed</code> by run this command:
```bash
bunx prisma db seed
```

Then run the server:
```bash
bun run server.ts
```
